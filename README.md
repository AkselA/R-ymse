# ymse
**ymse** [y`msə] 
adjective

	Various, of various kind, not specified

Old Norse *ýmsir*, *ýmissir*, pl. *ýmiss*, *ímiss*  

### usage
    library(devtools)
    install_github("AkselA/R-ymse")
    library(ymse)

    ?ymse::means
